- **index.html** - 中文版平板借還登記表

### 結果頁面

## 使用方法

### 1. 解壓縮檔案
將下載的zip檔案解壓縮到您的網站目錄

### 2. 上傳到伺服器
- 可直接上傳到GitHub Pages
- 可部署到任何網頁伺服器
- 支援本地測試

### 3. 配置GAS (如需要)
檔案中已配置Google Apps Script端點：
